## 📖 About

**VietSalePro V7** is a modern retail management platform developed to support small and medium-sized businesses in managing:

- Sales (POS)
- Inventory
- Purchasing
- Suppliers
- Customers
- Promotions
- Loyalty Program
- Reports & Analytics
- Multi-device synchronization
- Offline-first architecture
- Cloud synchronization

The project is continuously developed with a strong focus on:

- High performance
- Clean architecture
- Modern UI/UX
- Long-term maintainability
- AI-assisted development
- Design System driven development
